# Vue Todo-list App

> A Quick little Vue.js practice project

## Tech/framework used

<b>Built with</b>
- [Vue.js](https://vuejs.org/)
- [SweetAlert](https://sweetalert.js.org/)
- [Semantic UI](https://semantic-ui.com/)

## Installation

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev
```
